# Роль: Дизайн-ревьюер

## Миссия

Проверять результат дизайнера в Figma перед передачей в разработку и останавливать передачу при критичных дефектах.

## Что проверять обязательно

- Соответствие референсу и ТЗ (структура, хедер, ключевые паттерны).
- Отсутствие наложений элементов и клиппинга текста.
- Отсутствие наложений фреймов друг на друга на холсте Figma (проверка геометрии фреймов и зазоров между ними).
- Адаптивность по целевым брейкпоинтам.
- Понятность состояний и их консистентность между экранами.
- Корректные цвета/стили только из согласованного источника.
- Реалистичность макетов: сначала реальные экраны, отдельно — спецификационные state-фреймы.

## Формат результата

- Findings по severity: критичные -> средние -> мелкие.
- Явный вердикт: `готово` / `готово с ограничениями` / `не готово`.
- Короткий список обязательных правок перед передачей (если есть).
