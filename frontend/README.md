# Frontend — Easy English (Telegram Mini App)

Клиент: React 19, TypeScript, Vite, SCSS. Работает в Telegram WebView и в браузере (посадочная + авторизация через Telegram).

- **Запуск:** `npm run dev`
- **Сборка:** `npm run build`
- **Переменные:** `VITE_API_URL` — хост API (например, `http://localhost:3000/`). Пути API с префиксом `/v1` — см. [docs/server/api/README.md](../docs/server/api/README.md).
