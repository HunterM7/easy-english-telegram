# Настройки

[Docs](../../README.md) → [Client](../README.md) → [Screens](README.md) → **Settings**

**Когда показывается:** по нажатию пункта **«Настройки»** в нижнем меню.

**Назначение:** статистика пользователя, достижения и настройки приложения (в т.ч. автопрослушивание озвучки).

---

## Содержимое

- **Статистика:** уроки (пройдено), упражнения (выполнено), слова (выучено), дни (серии, всего дней с тренировками) — по мере реализации метрик.
- **Раздел достижений** — список всех достижений (полученные — в полном цвете, не полученные — приглушённо). Структура карточки: картинка, название, описание, награда (зернышки), дата получения (для полученных). Список достижений — в PRD, раздел 3.5.
- **Настройка автопрослушивания** — включить/выключить автоматическое воспроизведение озвучки после правильного ответа. Та же настройка доступна на экране после правильного ответа в тренажёре (PRD 3.8).
- **Цветовая тема** — три варианта: **Автоматическая** (тема из настроек Telegram), **Тёмная**, **Светлая**. При «Автоматическая» используем `themeParams` Telegram; при ручном выборе тёмной или светлой — сохраняем **локально на устройстве** (localStorage), чтобы тема на одном устройстве не меняла тему на другом. См. PRD, раздел 6.5.
- **Выйти (только в браузерной версии):** кнопка «Выйти» внизу экрана настроек. По нажатию — отправка `POST /v1/auth/logout` с текущим refresh-токеном (сервер инвалидирует сессию), очистка токенов на клиенте, переход на посадочную страницу. Нужна для смены аккаунта: пользователь может выйти и войти через другой Telegram. В Mini App кнопку можно не показывать (контекст уже привязан к одному аккаунту).

При необходимости: подписка, экспорт данных, удаление аккаунта — по решениям из PRD, раздел 14.
