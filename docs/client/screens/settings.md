# Настройки

[Docs](../../README.md) → [Client](../README.md) → [Screens](README.md) → **Settings**

**Когда показывается:** по нажатию пункта **«Настройки»** в нижнем меню.

**Назначение:** профиль пользователя, доступ к достижениям, статистика и настройки приложения.

---

## Структура экрана

### 1. Профиль
- Аватар пользователя (из Telegram)
- Имя пользователя
- Баланс зёрнышек (опционально, если не выводится только на главной)

### 2. Достижения
- **Кнопка «Достижения»** — переход на экран [Достижения](achievements.md)
- **Бадж (пузырь уведомлений)** — показывает количество достижений, готовых к активации
  - Если есть неактивированные достижения — бадж с числом (например, «3»)
  - Если нет — бадж не показывается

### 3. Статистика
Полный список метрик (см. PRD 3.5):
- **Прогресс:** тем завершено, уроков пройдено, упражнений выполнено
- **Обучение:** слов выучено, фраз составлено, правильных ответов (%), ошибок
- **Активность:** серия дней (текущая), максимальная серия (рекорд), всего дней с тренировками, дней с попугайчиком
- **Геймификация:** уровень попугайчика, общий XP, достижений получено (X из 25), всего зёрнышек заработано

### 4. Подписка
- **Статус подписки:**
  - Если активна: «Подписка активна до [дата]»
  - Если нет: «Бесплатная версия»
- **Кнопка действия:**
  - Если нет подписки: «Оформить подписку» → переход к оплате
  - Если есть: «Управление подпиской» или «Продлить»

### 5. Настройки приложения
- **Автопрослушивание** — переключатель (вкл/выкл). Автоматическое воспроизведение озвучки после правильного ответа. Та же настройка доступна в тренажёре (PRD 3.8).
- **Цветовая тема оформления** (не путать с учебной «темой» в смысле раздела курса — см. [PRD, глоссарий](../../product/PRD.md#11-глоссарий)) — три варианта:
  - **Автоматическая** — оформление из настроек Telegram (`themeParams`)
  - **Тёмная** — ручной выбор
  - **Светлая** — ручной выбор
  - Сохраняется **локально** (localStorage) — разные устройства могут иметь разные настройки.

## Состояния экрана

- **Загрузка:** скелетон страницы
- **Ошибка загрузки:** заглушка с кнопкой «Обновить»

---

## Адаптивность

- **Мобильный:** секции вертикально, на всю ширину
- **Планшет/десктоп:** возможно ограничение ширины контента, больше отступов
