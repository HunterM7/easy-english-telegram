# Содержимое темы (лекция + уроки)

[Docs](../../README.md) → [Client](../README.md) → [Screens](README.md) → **Theme content**

**Когда показывается:** после выбора темы из списка тем.

**Назначение:** показать пункт «Лекция» и список уроков внутри выбранной темы; пользователь может открыть лекцию (теория) или любой урок (тренажёр).

---

## Содержимое

- **Пункт «Лекция»** — переход на экран лекции по теме (грамматика, таблицы, объяснения).
- **Список уроков** темы — например: «Новые слова», «Прошедшее простое время», «Настоящее простое время» и т.д.

Пользователь может открыть лекцию или любой урок в произвольном порядке.

---

## Навигация

- **Лекция** → [Лекция по теме](lecture.md).
- **Урок (клик)** → [Тренажёр](trainer.md).
- **Назад** → [Список тем](themes-list.md).

## Состояния

- **Загрузка:** скелетон (список лекция + уроки).
- **Ошибка загрузки:** заглушка с кнопкой «Обновить».
