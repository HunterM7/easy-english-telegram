# Список тем

[Docs](../../README.md) → [Client](../README.md) → [Screens](README.md) → **Themes list**

**Когда показывается:** по нажатию пункта **«Темы»** в нижнем меню.

**Назначение:** показать все темы (например, Простое время, Герундий, Пассивный залог); по нажатию — переход к содержимому темы (лекция + уроки).

---

## Содержимое

- Список **тем** с названиями и при необходимости кратким описанием.
- По нажатию на тему — переход на экран **Содержимое темы** (лекция + список уроков).

---

## Навигация

- **Тема (клик)** → [Содержимое темы](theme-content.md).
- Нижнее меню: **Темы** (текущий экран) | иконка попугая → [Главная](home.md) | **Настройки** → [Настройки](settings.md).

---

## Состояния

- **Загрузка:** скелетон списка тем.
- **Ошибка загрузки:** заглушка с текстом об ошибке сети и кнопкой «Обновить».
