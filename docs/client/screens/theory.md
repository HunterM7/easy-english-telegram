# Теория темы

[Docs](../../README.md) → [Client](../README.md) → [Screens](README.md) → **Theory**

**Когда показывается:** при выборе пункта «Теория» из содержимого темы.

**Назначение:** показать всю грамматику и теорию по теме — объяснение, таблицы, примеры. Теория для всех уроков темы в одном месте. Только контент, без интерактивных заданий.

---

## Структура экрана

### Шапка
- **Кнопка «Назад»** — возврат к содержимому темы
- **Название** — «Теория: [название темы]» или просто название темы

### Контент теории
- Текст (параграфы, заголовки)
- Таблицы (спряжение, формы глаголов и т.д.)
- Картинки / иллюстрации (при необходимости)
- Примеры с переводом
- **Ссылки на уроки** — внутри текста могут быть ссылки для перехода к конкретному уроку темы (например, «Попробуй упражнения → [Прошедшее время]»)

Контент прокручивается вертикально. Оглавление не используется — теория представляет собой единый поток.

---

## Навигация

- **Назад** → [Содержимое темы](theme-content.md)
- **Ссылка на урок (внутри текста)** → [Тренажёр](trainer.md) соответствующего урока

---

## Состояния экрана

- **Загрузка:** скелетон контента (текстовые блоки)
- **Ошибка загрузки:** заглушка с кнопкой «Обновить»

---

## Адаптивность

- **Мобильный:** контент на всю ширину с отступами
- **Планшет/десктоп:** ограничение ширины контента для удобства чтения (не слишком широкие строки)
