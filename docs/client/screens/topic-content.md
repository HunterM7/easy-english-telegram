# Содержимое темы (теория + уроки)

[Docs](../../README.md) → [Client](../README.md) → [Screens](README.md) → **Содержимое темы**

**Когда показывается:** после выбора темы из списка тем.

**Назначение:** показать теорию по теме и список уроков; пользователь может открыть теорию (общая теория по теме) или урок (тренажёр с упражнениями).

---

## Структура экрана

### Шапка
- **Кнопка «Назад»** — возврат к списку тем
- **Название темы**

### Список содержимого

Единый вертикальный список, включающий:

1. **Теория по теме** — первый элемент списка
   - Отличается иконкой (книга)
   - Название: «Теория»
   - Краткое описание (опционально)

2. **Уроки темы** — остальные элементы списка
   - Название урока
   - Краткое описание (1–2 предложения)
   - Иконка, отличная от теории (или без иконки)

### Состояния элементов списка

| Состояние | Описание | Визуал |
|-----------|----------|--------|
| **Доступен** | Можно открыть | Обычный вид, кликабелен |
| **Заблокирован** | Предыдущий урок не завершён | Приглушённый (disabled), иконка замка |
| **Завершён** | Все упражнения пройдены | Обычный вид + галочка или метка |

**Правило разблокировки:** каждый следующий урок открывается после завершения предыдущего (линейная прогрессия внутри темы). Теория доступна всегда.

---

## Теория урока и теория по теме

- **Теория по теме** — общая теория по всей теме (грамматика, таблицы, примеры). Открывается с этого экрана.
- **Теория урока** — краткая теория конкретного урока. Показывается при первом открытии урока (перед упражнениями) и доступна по кнопке в тренажёре. См. [trainer.md](trainer.md).

---

## Навигация

- **Теория (клик)** → [Теория темы](theory.md)
- **Урок (клик на доступный)** → [Тренажёр](trainer.md) (при первом открытии — сначала теория урока)
- **Урок (клик на заблокированный — предыдущий не завершён)** → тултип «Завершите предыдущий урок»
- **Урок (клик на заблокированный — требуется подписка)** → всплывающий баннер подписки (аналогично списку тем)
- **Назад** → [Список тем](topics-list.md)

---

## Состояния экрана

- **Загрузка:** скелетон списка (теория + уроки)
- **Ошибка загрузки:** заглушка с кнопкой «Обновить»

---

## Адаптивность

- **Мобильный:** карточки на всю ширину
- **Планшет/десктоп:** увеличенные отступы, возможно ограничение ширины контента
