# Лекция по теме

[Docs](../../README.md) → [Client](../README.md) → [Screens](README.md) → **Lecture**

**Когда показывается:** при выборе пункта «Лекция» из содержимого темы.

**Назначение:** показать всю грамматику и теорию по теме — объяснение, таблицы, примеры. Только контент, без заданий. Всё, что нужно, чтобы понимать, как решать задачи в уроках этой темы.

---

## Содержимое

- Заголовки, параграфы, таблицы (спряжение, формы глаголов и т.д.), примеры.
- Только текст и разметка, без интерактивных заданий.

---

## Навигация

- **Назад** → [Содержимое темы](theme-content.md) (список лекция + уроки).

---

## Состояния

- **Загрузка:** скелетон контента лекции.
- **Ошибка загрузки:** заглушка с кнопкой «Обновить».
