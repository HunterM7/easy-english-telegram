# Тренажёр (страница урока)

[Docs](../../README.md) → [Client](../README.md) → [Screens](README.md) → **Trainer**

**Когда показывается:** при выборе урока из содержимого темы. Пользователь открывает урок → попадает в тренажёр (упражнения этого урока).

**Назначение:** показывать по одному заданию за раз; пользователь выполняет, отправляет ответ, получает проверку; после успешной проверки запрашивается следующее задание (логика — в PRD 3.4.1, API — в [api/trainer.md](../../server/api/trainer.md)).

---

## Содержимое

- **Шапка:** название урока, кнопка **«Назад»** (выход в список уроков темы). Подтверждение при выходе не показывается — прогресс сохраняется по каждому выполненному упражнению на сервере.
- **Область задания:** вопрос/подсказка, поле для ответа (сбор предложения, выбор варианта и т.д.), кнопки «Подсказка», «Удалить последнее слово» (для типа «сбор предложения») и т.п.
- **Футер:** прогресс в процентах (до первого прохождения урока — % заданий урока; после — общее число выполненных упражнений в уроке).
- Опционально: краткий блок теории перед заданиями.

После правильного ответа: позитивное сообщение, озвучка (если включено автопрослушивание), кнопка «Прослушать ещё раз» и переключатель автопрослушивания. После ошибки: показ правильного ответа, кнопка «Продолжить» / «Следующее задание».

---

## Навигация

- **Назад** → [Содержимое темы](theme-content.md).
- После последнего непройденного задания урока → [Урок завершён](lesson-completed.md).

---

## Состояния

- **Загрузка задания:** скелетон области задания.
- **Отправка ответа:** спиннер в кнопке «Проверить» (не блокировать весь экран).
- **Ошибка проверки/сети:** сообщение в контексте, кнопка «Отправить снова» без потери ввода (PRD 4.3).
- **Ошибка загрузки экрана:** заглушка с кнопкой «Обновить».
