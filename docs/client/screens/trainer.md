# Тренажёр (страница урока)

[Docs](../../README.md) → [Client](../README.md) → [Screens](README.md) → **Тренажёр**

**Когда показывается:** при выборе урока из содержимого темы. Пользователь открывает урок → попадает в тренажёр (упражнения этого урока).

**Назначение:** показывать по одному упражнению за раз; пользователь выполняет, отправляет ответ, получает проверку; после успешной проверки запрашивается следующее упражнение (логика — в PRD 3.4.1, API — в [api/trainer.md](../../server/api/trainer.md)).

---

## Теория урока (первое открытие)

При **первом открытии урока** (пользователь ещё не начинал этот урок):

1. Показывается **экран теории урока**:
   - Краткая теория по теме урока (правило, примеры, таблица — если нужно)
   - Кнопка **«Начать»** — переход к упражнениям

2. После нажатия «Начать» — загружается первое упражнение.

При **повторном открытии урока** — сразу загружается упражнение (теория не показывается автоматически, но доступна по кнопке).

---

## Структура экрана (тренажёр)

### Шапка
- **Кнопка «Назад»** — выход в список уроков темы (без подтверждения; прогресс сохраняется автоматически)
- **Название урока**
- **Кнопка «Теория»** (иконка книги / лампочки) — открывает теорию урока в модалке или отдельном экране

### Область задания
- Вопрос / подсказка на русском
- Поле для ответа (зависит от типа задания):
  - Сбор предложения: слова-кнопки + область собранного ответа + «Удалить последнее слово»
  - Выбор варианта: варианты ответов (A/B/C)
  - Ввод слова: текстовое поле
- Кнопка **«Подсказка»** (с указанием стоимости, если за зёрнышки)
- Кнопка **«Проверить»**

### Футер
- **Прогресс:**
  - До первого прохождения урока — процент выполненных заданий (например, «42%»)
  - После завершения урока (свободный режим) — общее число выполненных упражнений в уроке

---

## После ответа

### Правильный ответ
- Позитивное сообщение (визуально и/или текстом)
- **Озвучка** правильного ответа (если включено автопрослушивание)
- Кнопка **«Прослушать ещё раз»**
- Переключатель **автопрослушивания** (можно включить/выключить прямо здесь)
- Кнопка **«Дальше»** — запрос следующего упражнения

### Неправильный ответ
- Сообщение об ошибке
- Показ **правильного ответа**
- Кнопка **«Продолжить»** / **«Следующее задание»**

---

## Навигация

- **Назад** → [Содержимое темы](topic-content.md)
- **Теория** → модалка или экран с теорией урока
- **Дальше (после правильного)** → следующее упражнение
- После последнего непройденного упражнения урока → [Урок завершён](lesson-completed.md)

---

## Состояния экрана

- **Загрузка задания:** скелетон области задания
- **Отправка ответа:** спиннер в кнопке «Проверить» (не блокировать весь экран)
- **Ошибка проверки/сети:** сообщение в контексте, кнопка «Отправить снова» без потери введённого ответа (PRD 4.3)
- **Ошибка загрузки экрана:** заглушка с кнопкой «Обновить»

---

## Адаптивность

- **Мобильный:** компактная шапка, область задания на всю ширину, кнопки крупные (удобно нажимать)
- **Планшет/десктоп:** больше пространства вокруг области задания, возможно ограничение ширины контента
