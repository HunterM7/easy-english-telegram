# Посадочная страница

[Docs](../../README.md) → [Client](../README.md) → [Screens](README.md) → **Landing**

**Когда показывается:** только при открытии приложения по ссылке в **браузере** (вне Telegram). В Mini App внутри Telegram пользователь сразу попадает на главную, этот экран не показывается.

**Назначение:** описание приложения и вход через Telegram.

---

## Содержимое

- Краткое описание приложения: как работает, что есть маскот (попугайчик), темы, уроки, тренажёр.
- Кнопка **«Войти через Telegram»** (или «Open in Telegram» / «Login with Telegram») — в углу или по центру.

После успешной авторизации пользователь попадает в тот же интерфейс, что и при открытии в Telegram (главная → темы → уроки → тренажёр). Реализация: редирект в Telegram на бота/Mini App или Telegram Login Widget с последующим открытием приложения.

---

## Навигация

- **Войти через Telegram** → авторизация → переход на главную (страницу попугая).

---

## Состояния

- **Ошибка входа / нет Telegram:** сообщение в контексте и кнопка повтора (см. PRD, раздел 4.3).
