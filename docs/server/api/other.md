# API: Прочие эндпоинты (уточнить при реализации)

[Docs](../../README.md) → [Server](../README.md) → [API](README.md) → **Прочие эндпоинты**

Список эндпоинтов, которые планируются или уточняются на этапе реализации. Контракт и коды ответов — по мере готовности.

| Метод и путь | Назначение |
|--------------|------------|
| `GET /v1/themes/:topicId` | Содержимое темы (теория + список уроков). |
| `GET /v1/themes/:topicId/lessons` или `GET /v1/lessons?topicId=...` | Список уроков темы. |
| `GET /v1/users/me` или `GET /v1/profile` | Профиль пользователя, прогресс, настройки (в т.ч. автопрослушивание). |
| `PATCH /v1/users/me/settings` | Обновление настроек. |

При добавлении контракта создавать отдельный файл в [api/](README.md) (например, `profile.md`, `themes-detail.md`) или дополнять существующие.
